# Thallusin biosynthetic gene prevalence across mOTUs-db genomes

This repository contains the custom R script and associated input/output files used to analyze the distribution of thallusin-associated biosynthetic genes across 124,295 representative prokaryotic genomes from mOTUs-db.

## Overview

Translated mOTUs-db genomes were searched with MMseqs2 `easy-search` against nine target proteins associated with thallusin biosynthesis.

- EboA–F from *Maribacter stanieri* DSM 19891 (GCF_900112245.1)
- Ino1–3 from *Saccharomonospora* sp. CNQ-490 (GCF_000527075.1)

The R script parses the MMseqs2 output, identifies putative thallusin biosynthetic gene clusters, summarizes gene presence/absence across genomes and phyla, and prepares output files for visualization in iTOL.

## Repository structure

```text
scripts/
  thallusin_analysis.R

input_tables/
  Input tables required by the script

output_tables/
  Tables generated by the script

output_figures/
  Figures generated by the script

ebo_proteins/
  Target protein sequences used for the MMseqs2 search
```

## Large input file not included

The following file is required but not included because it is larger than 1 GB:

```text
input_tables/mOTUsv4.0_genome_summary.tsv
```

It can be downloaded from mOTUs-db:

```text
http://www.motus-db.org
```

Alternatively, it is available here:

```text
https://polybox.ethz.ch/index.php/s/4W7JtH37ArP68cE
```

After downloading, place the file in:

```text
input_tables/mOTUsv4.0_genome_summary.tsv
```

## Reproducing the R analysis

The analysis was run with R v4.5.3. To recreate the conda environment:

```bash
conda env create -f environment.yml
conda activate r_env
```

Then run the analysis from the repository root:

```bash
Rscript scripts/thallusin_analysis.R
```

## Main outputs

The script generates summary tables and figures in:

```text
output_tables/
output_figures/
```

It also generates files used for visualization of order-level prevalence of the core genes `eboB`, `eboC`, `eboE`, and `eboF` on a pruned GTDB R220 tree, which can be imported in iTOL (https://itol.embl.de/):

```text
output_itol/
```
